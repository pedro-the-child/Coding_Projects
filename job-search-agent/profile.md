# Candidate Profile

## Target Roles
- Accepted titles: Data Analyst, Senior Data Analyst, Analytics Engineer, Product Analyst, Business Analyst
- Rejected titles: Data Engineer, Data Scientist, Business Intelligence Developer, Machine Learning Engineer, Intern, Junior, Director, VP, Lead
 
## Skills
### Expert (5+ years)
- SQL, Python, Excel, tableau, Looker, BigQuery, Power BI

### Proficient (2-4 years)
- Git, statistical analysis, Airflow Data pipelines, dbt in production environment, GA4 (event-based analytics)

### Familiar (< 2 years)
- Spark, Salesforce, Google Apps Script, K-means clustering, LLM application development (classification pipelines, NLP text processing at scale)

## Domain Experience
- A/B testing (promotions, marketing, email campaigns)
- Customer LTV and cohort analysis
- Self-service dashboard design
- Data pipeline experience
- LLM-queryable data product design (dbt semantic layer enrichment: column-level documentation, naming conventions, grain definitions enabling natural language querying across 300k+ campaigns)
- LLM integration: deployed LLM classification pipeline for unstructured text at scale (survey responses → sentiment signals)

## Hard Requirements
- Fully remote (quarterly in-office is ok)
- Seniority: Mid to Senior only
- Location: US only
- Compensation, title filters, and disqualifying requirements: see `eval/first_pass_rules.md`
