# Title & Source Filter

## How to Use This File
Titles are **not** a hard gate. Many roles with non-standard titles (e.g., "GTM Analyst", "Strategy Analyst") have job content that matches the candidate's profile. Use this file for:
- **Auto-reject:** titles/seniority in the hard reject list below (wrong function entirely)
- **Auto-accept:** titles in the accepted list below (no further title judgment needed)
- **Judgment call:** anything else — skim the listing content before deciding

## Accepted Titles (auto-pass title check)
- Data Analyst
- Senior Data Analyst
- Analytics Engineer
- Product Analyst
- Business Analyst

## Hard Reject Titles (wrong function, always skip)
- Data Engineer
- Data Scientist
- Machine Learning Engineer
- Software Engineer
- Deal Desk Analyst
- Intern (any title)
- Junior (any title)

## Titles That Require Content Review
These often appear in search results. Some are good fits, some aren't — read enough of the listing to judge:
- GTM Analyst / GTM Analytics Engineer — often a match if core work is SQL/dashboards/A/B testing
- BI Analyst / BI Developer — match if analytics-focused, reject if ETL/infrastructure-heavy
- Revenue Analyst / Revenue Ops — match if analytics, reject if systems admin
- Strategy Analyst — depends entirely on job content
- Planning Analyst — usually ops-focused, rarely a match

## Seniority
- **Accept:** Mid, Senior, IC, II, III
- **Reject:** Intern, Junior, Entry Level, Director, VP, C-level
- **Context-dependent:** Associate (accept if mid-level content), Lead (accept if IC, reject if people management), Principal (accept if IC analytics)

## Known Staffing Agencies / Aggregators (skip entirely)
- Jobgether
- Stott and May
- Prestige Staffing
- Swoon
- HARAMAIN
- Crossing Hurdles

These post on behalf of unnamed clients and should be filtered during search result processing, before any evaluation.
